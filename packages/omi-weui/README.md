## omi-weui

OMI 版本 WeUI 是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信内网页和微信小程序量身设计，令用户的使用感知更加统一。


## Installation

```bash
npm i omi-weui
```

## Usage

```tsx
import '@omi-weui/button'
```

在各个框架中使用:

```tsx
<w-button>按钮</w-button>  // omi
<w-button>按钮</w-button> // vue
<w-button>按钮</w-button> // html
```
